# Sistema de Agenda Inteligente para Mascotas

## Descripción

El Sistema de Agenda Inteligente para Mascotas es una solución diseñada para ayudar a los propietarios a gestionar de manera organizada la información relacionada con el cuidado de sus mascotas. El sistema permite registrar datos de las mascotas, programar citas, llevar control de vacunas y mantener un historial de salud actualizado.

## Objetivo del Proyecto

Desarrollar una herramienta que facilite el seguimiento de actividades importantes para el bienestar de las mascotas, reduciendo olvidos y permitiendo un mejor control de su información médica y administrativa.

## Funcionalidades Principales

- Registro y gestión de usuarios.
- Registro de mascotas.
- Administración de información de cada mascota.
- Programación de citas veterinarias y estéticas.
- Visualización de calendario de eventos.
- Control del historial médico.
- Gestión de vacunas y desparasitaciones.
- Generación de recordatorios automáticos.
