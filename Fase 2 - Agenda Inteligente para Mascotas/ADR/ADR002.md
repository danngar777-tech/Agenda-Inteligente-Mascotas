# ADR-002: Uso de MySQL como Base de Datos

## Estado

Aceptado

## Contexto

El sistema requiere almacenar información de usuarios, mascotas, citas, vacunas e historial médico.

## Decisión

Se utilizará MySQL como sistema gestor de base de datos.

## Justificación

MySQL es una solución ampliamente utilizada, estable y adecuada para aplicaciones de gestión de información.

## Consecuencias

- Almacenamiento estructurado de datos.
- Facilidad de administración.
- Disponibilidad de documentación y soporte.
